#### 将代码写的简洁并且易读易懂是每一位优秀的coder所应该具备的基本功。

#### 前几天在github上看到clean-code-php这个项目，感觉很有收获，于是在这里记录一下。

#### 使用有意义并且可读的变量名称

#### Bad:

```
$ymdstr = $moment->format('y-m-d');
```

#### Good:

```
$currentDate = $moment->format('y-m-d');
```
